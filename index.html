Getting Started

  1. Clone the repository

    git clone https://github.com/jakmaz/free-games-notifier.git
cd free-games-notifier

  2. Install dependencies

    npm install

  3. Set up configuration

    Edit the config/default.yaml file to set up your platform preferences and notification channels.

  4. Run the application

    npm build:start

Configuration

Adjust the config/default.yaml to match your preferences. Hereâ€™s an example for setting up Steam and using Ntfy for notifications:

  platforms:
